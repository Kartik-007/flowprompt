# Changelog

All notable changes to this project will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/), and this project adheres to [Semantic Versioning](https://semver.org/).

## [1.0.0] - 2026-02-26

### Added

- Global hotkey launcher (`Ctrl+Cmd+P`) with Spotlight-style floating panel
- Quick save (`Ctrl+Cmd+S`) to capture selected text or clipboard as a new prompt
- Auto-paste into the active text field with clipboard preservation
- Fuzzy search across title, content, tags, and categories
- Full keyboard navigation with arrow keys, Enter, and Esc
- Collapsible category organization
- Recently used prompts (last 5)
- Prompt editor with title, content, category, and tags
- Human-readable JSON storage at `~/.flowprompt/prompts.json`
